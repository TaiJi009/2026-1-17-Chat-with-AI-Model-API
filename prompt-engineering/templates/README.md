# 提示词模板库

本文件夹用于存放可复用的提示词模板。

## 📁 目录结构

```
templates/
├── README.md              # 本文件，使用说明
├── system-prompts/        # 系统提示词模板
├── user-prompts/          # 用户提示词模板
└── examples/              # 示例模板
```

## 🎯 使用方式

### 系统提示词模板

系统提示词定义了 AI 的角色、能力和行为规范。

**模板位置**：`templates/system-prompts/`

**使用步骤**：
1. 选择一个合适的系统提示词模板
2. 根据具体需求进行定制
3. 在项目的 `PromptPanel` 组件中配置使用

### 用户提示词模板

用户提示词模板定义了用户输入的格式和变量替换规则。

**模板位置**：`templates/user-prompts/`

**变量说明**：
- `{{user_input}}` - 用户输入的消息
- `{{date}}` - 当前日期
- `{{time}}` - 当前时间

### 示例模板

包含各种场景下的完整示例，可直接参考使用。

**模板位置**：`templates/examples/`

## 📝 模板格式

每个模板文件应包含以下信息：

```markdown
# 模板名称

## 用途
简要说明该模板的用途和适用场景

## 模板内容
```
（实际的提示词内容）
```

## 变量说明
- `{{variable_name}}` - 变量说明

## 使用示例
示例：如何使用该模板

## 注意事项
任何需要注意的事项
```

## 🔄 贡献新模板

1. 在相应的子文件夹中创建新的 Markdown 文件
2. 按照上述格式编写模板文档
3. 在本 README 中更新目录说明

---

**提示**：模板应该简洁明了，易于理解和修改。
