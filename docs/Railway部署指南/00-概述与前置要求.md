# 概述与前置要求

## 概述

Railway 是一个现代化的应用部署平台，支持：

- ✅ 自动检测 Node.js 项目，无需 Dockerfile
- ✅ 自动提供 PostgreSQL 数据库
- ✅ GitHub 集成，代码推送自动部署
- ✅ 自动 SSL 证书和 HTTPS
- ✅ 简单易用的环境变量管理
- ✅ 实时日志查看
- ✅ 支持自定义域名

## 前置要求

在开始部署之前，请确保您已准备好以下内容：

- **Railway 账号**：访问 [railway.app](https://railway.app) 注册
- **GitHub 账号**：用于代码仓库和自动部署
- **项目代码已推送到 GitHub 仓库**：确保代码已提交并推送到 GitHub
- **微信支付配置信息**（如果使用支付功能）：
  - App ID
  - 商户号 (MCH ID)
  - API Key
  - 回调 URL
- **腾讯云短信服务配置信息**（如果使用短信功能）：
  - Secret ID
  - Secret Key
  - 应用 ID
  - 签名名称
  - 模板 ID
