## 概述

Railway 是一个现代化的应用部署平台，支持：

- ✅ 自动检测 Node.js 项目，无需 Dockerfile
- ✅ 自动提供 PostgreSQL 数据库
- ✅ GitHub 集成，代码推送自动部署
- ✅ 自动 SSL 证书和 HTTPS
- ✅ 简单易用的环境变量管理
- ✅ 实时日志查看
- ✅ 支持自定义域名
