# 03-启动服务

## 步骤说明

这是快速开始的第三步：启动 Docker 容器服务。

## 启动服务

1. 进入 Docker 配置目录：

```bash
cd docker
```

2. 启动所有服务（后端和数据库）：

```bash
docker-compose up -d
```

参数说明：
- `-d`：后台运行（detached mode）

## 首次启动

首次运行时会执行以下操作：
- 下载所需的 Docker 镜像（PostgreSQL、Node.js 等）
- 构建后端应用镜像
- 创建并启动容器
- 初始化数据库

这个过程可能需要几分钟时间，请耐心等待。

## 查看启动状态

启动后可以查看服务状态：

```bash
docker-compose ps
```

应该看到两个服务在运行：
- `chat_app_postgres`（数据库）
- `chat_app_backend`（后端API）

## 查看日志

如果需要查看启动日志：

```bash
# 查看所有服务日志
docker-compose logs -f

# 只查看后端日志
docker-compose logs -f backend

# 只查看数据库日志
docker-compose logs -f postgres
```

按 `Ctrl+C` 退出日志查看。
