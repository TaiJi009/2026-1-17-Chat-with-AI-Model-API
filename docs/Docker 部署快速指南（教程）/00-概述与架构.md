# Docker 部署快速指南

## 概述

本指南提供使用Docker部署账号系统和付费系统的快速步骤，适用于本地开发和生产环境。

## 部署架构

```
┌─────────────────┐
│   前端 (GitHub Pages) │
└────────┬────────┘
         │ HTTPS
         ▼
┌─────────────────┐
│  Docker容器 (后端API) │
│  Express + TypeScript │
└────────┬────────┘
         │
    ┌────┴────┐
    ▼         ▼
┌────────┐ ┌──────────┐
│PostgreSQL│ │外部API服务│
│(Docker) │ │微信支付/短信│
└────────┘ └──────────┘
```
